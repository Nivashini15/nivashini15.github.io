Sentimental Alpha

Sentimental Alpha is an end-to-end machine learning pipeline that merges quantitative time-series analysis with deep learning-based NLP for End-of-Day (EOD) stock forecasting. By utilizing Hugging Face's S-BERT to extract 384-dimensional semantic embeddings from live financial news and fusing them with complex technical indicators (MACD, RSI), this hybrid Random Forest model delivers highly accurate, Explainable AI (XAI) predictions through a responsive Streamlit dashboard.

RAG Document Q&A

Intelligent Document Q&A System is an end-to-end Retrieval-Augmented Generation (RAG) pipeline that merges high-dimensional vector search with advanced open-source Large Language Models for dynamic knowledge extraction. By utilizing LangChain and Hugging Face models to extract deep semantic embeddings from unstructured documents, and indexing them within a highly optimized FAISS vector database, this context-aware AI architecture delivers highly accurate, fact-grounded responses. The system effectively eliminates model hallucinations, ensuring precise, real-time semantic querying through an interactive conversational interface.
